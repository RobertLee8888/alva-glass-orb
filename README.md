# Alva Glass Orb

Alva AI 的动态品牌形象 —— 一颗有生命的液态玻璃球。Alva 的 mark 悬浮在球心，四种状态用完全不同的「表情」表达 AI 的工作节奏，所有效果由单文件 WebGL 片元着色器实时渲染，无任何外部依赖。

**🔮 在线 Demo：<https://robertlee8888.github.io/alva-glass-orb/>**

## 四种状态

| 状态 | 动效 |
|------|------|
| **待机** | 中性基线：局部褶皱缓慢漂移、呼吸幅度 <1%、边缘光彩约 30s 环流一圈；每 2~6s 随机一次柔和的色散闪或轻吸气，节奏不可预测 |
| **思考** | 整球尺寸抖一下，mark 弹射成六个 logo 同款圆角方块绕心转圈（慢-快-慢甩动、逐个自翻转）；边缘光彩被卷着滞后半拍加速、向外圈聚拢（离心感），退出带惯性滑落 |
| **输出中** | 圆点汇成笔尖，以四种随机笔迹之一疾书手写「Alva」签名（约 1.5s 一气呵成）；墨迹留痕、新墨深旧墨淡、笔触推开周围光晕；写完按书写顺序逐段消散，顿一拍换一种笔迹重写 |
| **等待回应** | 球外右上一枚呼吸的蓝点示意「该你了」；每 10s 先向后蓄力、再迪士尼式疾速绕球一圈——轨迹像绳子把球勒出腰身，松绳弹回，蓝点归位时球已复原 |

## 交互

表情归内容、交互归球：待机的呼吸 / 眨眼 / 随机张望由球心 mark 承载；hover 时球轻微放大、朝鼠标一侧凹陷、色带向反方向退让、脸转向鼠标；按下像盖玻片压液滴——摊开变大、球顶压平、内容放大、投影消失，按住保持；抬起弹簧回弹带过冲。

## Liquid Glass

切换到「页面内容」视图：背景变成缓慢上滚的正常字号段落，模拟按钮常驻页面右下角时用户滚动页面的场景。球后的文字被**折射**（采样坐标沿球面法线弯折，边缘更强）与**模糊**（mipmap LOD 偏置多次采样），呈现 iOS 液态玻璃的光学质感。

## 展示规格

- **PC 常驻按钮**：48px（对齐 Notion 右下角气泡量级）
- **Chat 行内 logo**：32px（对齐 Claude 行内头像量级）
- 三种配色场景：白底·黑绿 / 深底·白绿 / 品牌绿·黑白，每种场景独立调校迷幻色板、色散强度、投影与示意点颜色

## 技术要点

- 单文件 HTML + WebGL1 片元着色器，Alva 官方 mark（SVG）与手写签名路径（Catmull-Rom + 弧长重采样的 1D 纹理 ×4 变体）全部内嵌
- 有机感来源：非匀速节奏（无理数比频率叠加）、间歇性随机微事件、状态切换的预备-过冲-回弹（anticipation / overshoot / settle）、细节间的因果联动（色彩环流被圆点「卷」着走、笔触推开光晕）
- 水滴式墙面投影（贴边暗线 / 透镜亮隙 / 本影 / 半影四层月牙）、真实色散（RGB 通道分离）、前后两层视差迷幻色
